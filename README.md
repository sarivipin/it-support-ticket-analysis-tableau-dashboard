# IT Support Data Analysis Dashboard

## 📌 Project Overview
This project analyses IT support ticket data to identify trends, performance bottlenecks, and operational insights. The goal is to support data-driven decision-making by visualising key metrics such as ticket volume, resolution time, and ticket distribution across product areas.

---

## 📊 Dashboard Features
- Ticket volume trend over time
- Tickets by product area
- Resolution time analysis by priority
- Status breakdown (open, in progress, resolved, etc.)
- Interactive Tableau dashboard

---

## 🛠️ Tools & Technologies
- Tableau (Dashboard & Visualisation)
- Python (Pandas) *(for data analysis / preprocessing)*
- CSV Dataset
- GitHub (Version Control)

---

## 📁 Dataset
- Source: Kaggle (Synthetic IT Support Ticket Dataset)
- The dataset includes:
  - Ticket ID
  - Created Date
  - Priority
  - Status
  - Product Area
  - Resolution Time

---

## 📈 Key Insights
- High priority tickets tend to have longer resolution times
- Certain product areas generate higher ticket volumes
- Majority of tickets are successfully resolved
- Ticket volume remains relatively consistent over time

---

## 🚀 How to Use
1. Download the dataset or use provided CSV
2. Open the Tableau dashboard file (.twbx)
3. Explore interactive visualisations
